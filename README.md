# Краткая инструкция(в Swagger)
1. Регистрируетесь выбирая одну из трёх ролей - User, Manager, Admin
2. Полученный Acess token введите в строку Authorize спарва вверху  
Полученный Refresh token пригодится вам позже
3. Если выбрали User - можете почитать публичные документы воспользовавшись Get запросом  
Если выбрали Manager или Admin, то вам необходимо связаться с разработчиком(Егором Макаровым) в вк  
Я выдам вам код подтверждения который будет дейстивтелен в течении 5 минут после запроса  
Полученный код вам необходимо будет отправить новым запросом  
В ответ вам придёт новый Acess token токен который вам необходимо будет так же ввести в строку Authorize спарва вверху
Полученный Refresh token пригодится вам позже
Manager может создавать документы, редактировать свои, видеть свои и публичные
Admin - всемогущий

# Полезные факты
При последующих входах вы можете просто логиниться.    
На каждый IP в час доступно лишь 5 запросов на регистрацию, 5 запросов на логин, 3 запроса на подтверждение кодом.
Для удобства тестирования я добавил endpoint который обнуляет лимит запросов сделанных в этом часу с этого IP  
Через час ваш Acess token устареет, чтобы не логиниться заного отправьте на endpoint refresh ранее полученый Refresh token.
В ответ вы получите новый Acess token.     
Rate Limiter реализовал не через бд, а через оперативную память т.к. нагрузка не большая сейчас и это быстрее работает.
